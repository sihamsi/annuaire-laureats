Full-stack geoinformation project — Spring Boot REST API + React frontend.

> 🚧 **Work in progress** — the repository currently holds the project skeleton; features are being built incrementally.

## Repository layout

| Folder | Description | Stack |
|---|---|---|
| [`backend/`](backend/) | REST API | Java · Spring Boot · Maven |
| [`frontend/`](frontend/) | Web client | React 19 · Create React App |

## Getting started

**Backend** (requires Java 17+):

```bash
cd backend
./mvnw spring-boot:run
```

**Frontend** (requires Node 18+):

```bash
cd frontend
npm install
npm start
```

The frontend runs on `http://localhost:3000`, the backend on `http://localhost:8080`.
